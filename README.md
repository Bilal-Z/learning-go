# Learning Go

## Go CLI

- `go build`

  compile sorce code file(s)

- `go run`

  compile and execute

- `go fmt`

  format all the code in dir

- `go install`

  compile and installs a package

- `go get`

  download package

- `go test`

  run tests

## Go packages

Types of packages:

- executable (main)

  generates a file we can run must have a main function

- reusable

  place to store reusable logic
